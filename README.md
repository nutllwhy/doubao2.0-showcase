# Doubao 2.0 Showcase

豆包2.0编程能力测试：展示四个网页项目的展示平台，包含两个小游戏和两个小工具。

## 项目列表

### 🎮 小游戏

1. **贪吃蛇豪华版** - 经典贪吃蛇，加入特效、道具、分数系统和本地排行榜
2. **2048 AI对战版** - 经典2048，加入AI对战功能和自动演示

### 🛠️ 小工具

3. **Markdown实时预览编辑器** - 分屏编辑、语法高亮、导出功能
4. **CSS渐变生成器** - 可视化编辑、多种渐变类型、预设方案、一键复制

## 技术栈

- 纯HTML/CSS/JavaScript（无需框架，直接在GitHub Pages运行）
- 响应式设计，支持各种屏幕尺寸

## 快速开始

直接用浏览器打开 `index.html` 即可！

或者访问 GitHub Pages：[https://nutllwhy.github.io/doubao2.0-showcase/](https://nutllwhy.github.io/doubao2.0-showcase/)

## 项目结构

```
doubao2.0-showcase/
├── index.html          # 主页
├── README.md          # 说明文档
├── games/
│   ├── snake/         # 贪吃蛇豪华版
│   └── 2048/          # 2048 AI对战版
└── tools/
    ├── markdown-editor/  # Markdown编辑器
    └── gradient-generator/ # CSS渐变生成器
```
