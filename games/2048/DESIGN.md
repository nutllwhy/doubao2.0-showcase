# 2048 AI对战版 - 设计文档

## 功能特性

### 核心玩法
- 经典2048游戏逻辑
- 方向键/WASD控制
- 平滑的方块移动动画

### AI对战模式
- 玩家 vs AI 同时进行
- 左右分屏显示
- AI使用贪心算法+启发式搜索
- 实时对比双方分数

### AI自动演示
- AI自动玩游戏
- 可调节速度
- 展示AI策略

### 统计功能
- 步数统计
- 游戏时间
- 最高分记录
- 最大方块记录

### 视觉效果
- 渐变方块配色
- 平滑移动动画
- 合并时的缩放效果
- 响应式设计

## 技术实现

### 游戏核心
- 4x4网格
- 方块合并逻辑
- 随机生成新方块（2或4）

### AI算法
- 启发式评估函数：
  - 单调性（方块按顺序排列）
  - 平滑性（相邻方块差值小）
  - 空格数（越多越好）
  - 最大方块位置（角落最好）
- 贪心搜索：每次选择评估最高的方向

### 动画系统
- CSS transitions
- requestAnimationFrame
- 方块位置追踪
