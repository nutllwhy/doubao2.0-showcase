# 贪吃蛇豪华版 - 设计文档

## 功能需求
- 经典贪吃蛇玩法
- 特效粒子系统（吃食物时的爆炸特效）
- 多样道具：加速、减速、穿墙、双倍分数
- 三种难度模式（简单/普通/困难）
- 本地排行榜（保存最高分）
- 流畅的动画和视觉效果

## 技术方案

### 文件结构
```
games/snake/
├── index.html   # 入口页面
├── style.css    # 样式
└── game.js      # 游戏逻辑
```

### 核心模块

1. **游戏画布 (Canvas)**
   - 20x20 网格，每个格子 25px
   - 总尺寸：500x500px

2. **蛇 (Snake)**
   - 数组存储身体坐标
   - 方向控制（上/下/左/右）
   - 自动移动

3. **食物 (Food)**
   - 随机生成位置
   - 不同颜色表示不同食物类型

4. **道具 (Power-ups)**
   - 加速：蛇速度变快
   - 减速：蛇速度变慢
   - 穿墙：可以穿过墙壁
   - 双倍分数：吃食物得双倍分
   - 随机时间生成

5. **粒子系统 (Particles)**
   - 吃食物时生成爆炸粒子
   - 粒子有不同颜色和速度
   - 逐渐淡出消失

6. **排行榜 (Leaderboard)**
   - 使用 localStorage 保存
   - 保存前10名最高分
   - 显示玩家名字和分数

7. **难度模式**
   - 简单：速度慢，道具多
   - 普通：中等速度，道具适中
   - 困难：速度快，道具少

## 游戏流程
1. 选择难度模式
2. 游戏开始，蛇自动移动
3. 玩家用方向键控制
4. 吃食物得分，吃道具获得效果
5. 撞墙或撞自己游戏结束
6. 输入名字，保存分数
7. 查看排行榜

## 配色方案
- 背景：深色渐变 (#1a1a2e → #16213e)
- 蛇头：亮绿色 (#00ff88)
- 蛇身：渐变绿色 (#00cc66 → #009944)
- 食物：红色 (#ff4757)、橙色 (#ffa502)、紫色 (#a55eea)
- 道具：金色 (#ffd700)、蓝色 (#3498db)
