# Markdown实时预览编辑器 - 设计文档

## 功能特性

### 核心编辑
- 分屏布局：左侧编辑，右侧预览
- 实时预览：输入时即时更新
- 语法高亮：Markdown语法高亮显示

### Markdown支持
- 标题（H1-H6）
- 粗体、斜体、删除线
- 列表（有序/无序）
- 链接和图片
- 代码块（支持多种语言高亮）
- 表格
- 引用
- 分隔线

### 额外功能
- 本地存储自动保存
- 导出HTML功能
- 导出PDF功能
- 清空编辑器
- 示例模板加载

### 视觉效果
- 现代化深色/浅色主题
- 流畅的动画过渡
- 响应式设计，移动端适配
- 代码高亮（Prism.js）

## 技术实现

### Markdown解析
- 使用marked.js库解析Markdown
- 配置支持GFM（GitHub Flavored Markdown）

### 代码高亮
- 使用Prism.js进行代码语法高亮
- 支持常见编程语言

### 本地存储
- 使用localStorage保存编辑器内容
- 自动保存（防抖处理）

### PDF导出
- 使用html2canvas + jsPDF
- 或者使用浏览器的打印功能
